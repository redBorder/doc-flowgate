3.2. External Configuration process

Once the configuration process is finished,the last step is to configure the client proxy in the redBorder Enterprise manager instance.
If all the above process has been done correctly, in the Sensors section of the manager, you must see the unclaimed sensor created:
Figure 1: External Configuration process
The next step is to create a Claim Sensor in the Redborder manager Enterprise edition.
Therefore, yo must press +Add Sensor and +Claim Sensor:
Figure 2: External Configuration process
A pop-up window will appear where you must enter a name for the sensor and paste the UUID that you copied before in the field below. 
Figure 3: External Configuration process
Click on  save and wait between 5-10 minutes until the sensor is configured correctly.
You must wait a few minutes to get a green indicator, and then, you can verify that the sensor is working in RedBorder Enterprise. 
Figure 4: External Configuration process
Sensor registered and claimed. 
Figure 5: External Configuration process
