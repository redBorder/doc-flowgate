1.4. Burning the ISO Image

The ISO image is prepared to perform the boot from a DVD reader, USB device or as an ISO file for a virtual machine.
In order to burn the ISO image from a Linux system to a USB device, the following command needs to be used if the USB device is mapped in /dev/sdd: 
[root@machine ~]# dd if=redBorder-3.1.XX-XX-x86_64-6.5-proxy.iso of=/dev/sdd bs=10M

Note

Replace XX-XX in redBorder-3.1.XX-XX-x86_64-6.5-proxy.iso by the patched version of the redborder 3.1 that you have, for example: redBorder-3.1.80-18-x86_64-6.5-proxy.iso.
