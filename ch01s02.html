Figure 1: Requirements for installing redBorder Flowgate

Warning

It is VERY IMPORTANT that the configuration of the Manager (or cluster of managers) is performed before configuring FlowGate sensors.
The Manager must be operational and in a network accessible to the sensors. Some of the sensor configuration processes depend on the access to the Manager.
It is also IMPORTANT that the FlowGate sensor network is operational during the first boot process. The FlowGate sensor will contact the Manager in order to perform a sensor registration.
